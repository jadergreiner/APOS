## 🚨 Riscos & Mitigações

### Risco 1: Personas Não Agendarem (🔴 ALTO)

**Impacto**: Perder entrevistas JTBD → não validar job → atrasar R1  
**Mitigação**: 
- Começar outreach HOJE (19 jul)
- Múltiplos canais (LinkedIn, Slack, email, referências)
- Ter 3 datas alternativas por persona
- Confirmar até 21 jul

### Risco 2: Sobrecarga de Esforço (🟡 MÉDIO)

**Impacto**: T0.0.1-3 (4d) + T0.0.A-C (4d) = 8d, mas só temos 5 dias  
**Mitigação**:
- Paralelizar T1 (Framework) + T2 (JTBD) a partir de 22 jul
- T1 não bloqueia T2 — podem rodar em paralelo
- Base em S0.1: velocidade real é 5 pts/dia, então 8 dias de trabalho = 1.6 dias reais

### Risco 3: Qualidade de Bootstrap Gate (🟡 MÉDIO)

**Impacto**: Gate com bugs → projetos não conseguem onboard → defeito crítico  
**Mitigação**:
- TDD approach obrigatório
- Tests unitários + integração antes de merge
- Code review antes de 26 jul
- Verificação de cobertura (target 80%+)

### Risco 4: Feedback de Job Statement Negativo (🟡 MÉDIO)

**Impacto**: Job statement não valida → R1 strategy muda → atrasos cascata  
**Mitigação**:
- Entrevistas com personas reais (não teóricas)
- Forces matrix para validar push/pull
- Multiple rounds de validação (T0.0.B, T0.0.C)
- Escalate para Jader se >30% das entrevistas contradizem job statement
